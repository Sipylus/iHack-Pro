# iHack Pro

Clover EFI Bootloader Theme

<img src="/Theme/screenshot.png">

# Installation
★ Download the iHack Pro theme and unpack the contents of the archive.<br />
★ Mount the ESP of your boot drive.<br />
★ Open EFI&gt;EFI&gt;CLOVER&gt;themes.<br />
★ Copy the unpacked iHack Pro theme and paste in the themes folder.<br />
★ Open the EFI&gt;EFI&gt;CLOVER&gt;config.plist file in an editor.<br />
★ Change the theme name to match iHack Pro.<br />
★ If this is your primary monitor, set the screen resolution to match, then save.<br />
★ Unmount the ESP to protect the contents.<br />
<br />
That&#8217;s it! When you restart your system, the iHack Pro theme will load and play the boot sound.

# Requirements
★ Clover EFI v2.4k r4945 or later<br />
